### Strings

Strings are a complex type that are made of array of chars with null terminator '\0', and by default is utf8 encoded:
```
str[8,16,32]
```
`str` is similar to `seq[char[]]`



### Imaginary
Imaginary numbers
```
img[]
```

### Optional
Either are empty or have a value
```
opt[]
```

### Multiple
```
mul[]
```

### Any
```
any[]
```
